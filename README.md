# AI Job Market Intelligence

An end-to-end data analytics project analyzing the global AI job market to identify hiring trends, in-demand skills, salary insights, and compare key metrics between Germany and the global market.

## Tools
- Power BI
- Python (Google Colab)
- Google Sheets

## Project Highlights
- Global AI job market analysis
- Skills demand analysis
- Salary and experience insights
- Germany vs Global comparison
- Interactive Power BI dashboard
